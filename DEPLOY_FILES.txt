## üìÅ Deployment Build Process

### Step 1: Clean Build
```bash
rm -rf node_modules dist
npm install
npm run build
```

### Step 2: Files in dist/ folder
After build, dist/ contains:
- index.html
- assets/*.js (bundled JavaScript)
- assets/*.css (bundled CSS)
- Static assets

### Step 3: Deploy dist/ folder
Upload only the dist/ folder to:
- Firebase Hosting
- Netlify
- Vercel
- GitHub Pages
